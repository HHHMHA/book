Requirements
------------

* Django 2.2+
* Python 3.6+

Installation
----------------------------

#. Clone the git repository.
#. Please check `Settings`_ section for configurations

#. Install all third party packages by running::

    $ pip install -r requirements.txt

#. Apply migrations::

    $ python manage.py migrate
